Consumption monitoring platform for blockchain development

Project providing a ready-to-use platform for monitoring and evaluating consumption on blockchain networks.

Main goal

Platform specifically focusing on the Ethereum mainnet and a private blockchain testbench. Provide accurate and actionable data on various on-chain & off-chain consumptions, enabling users to better understand and optimize their blockchain operations.

Key Components

  • Ethereum Mainnet Consumption Dashboard
    • Utilizes client distribution and scientific reports to estimate consumption.
    • Integrates API data from reputable services into interactive graphs.
    • Estimates idle network costs, transaction costs, and daily total costs.

  • Private Blockchain TestBench
    • Deploys a private blockchain using servers as physical machines connected to wattmeters.
    • Implements Grafana and Prometheus for detailed monitoring of both blockchain clients and Raspberry Pi devices.
    • Provides an interface to configure network parameters, create and manage smart contracts, and track on-chain and off-chain metrics.
    • Offers easy and reliable deployment with extensive error management and configuration options.

Benchmarking Large Language Models for Ethereum Smart Contract Development (OPEN-SOURCE)

Orange-OpenSource

This project created at Orange-Innovation, provides an automated pipeline to generate smart contracts using Large Language Models with Ollama. The pipeline compiles these smart contracts with the Solidity compiler (solc), analyzes them using Slither, and performs unit testing based on the provided test instructions in the prompts. Statistics are then produced to determine the efficiency of each model and various prompts are used to interpret the results.

Pipeline architecture


Built with

  • Ollama: Running large language models locally.
  • Slither, the smart contract static analyzer: Slither is a Solidity & Vyper static analysis framework written in Python3.
  • The solidity compiler: build by the Ethereum Foundation.
  • Hardhat: Hardhat is an Ethereum development environment for professionals, Built by the Nomic Foundation.

Pipeline created in python, running inside a Docker container.
